# xcsh for VS Code

> Manage F5 Distributed Cloud resources directly from VS Code

![xcsh](resources/screenshot-welcome.png)

## Getting Started

1. **Install the extension** — search for "xcsh" in the VS Code Extensions panel
2. **Install xcsh** — `brew install f5xc-salesdemos/tap/xcsh`
3. **Add a context** — open the Command Palette (`Cmd+Shift+P`) and run **xcsh:
   Add Context**

## What You Can Do

- **Browse and manage resources** — create, edit, delete load balancers, WAF
  policies, origin pools, and more from the sidebar
- **AI chat assistant** — ask `@xcsh` questions in natural language to manage
  your platform
- **Cloud status dashboard** — real-time global infrastructure health at a
  glance
- **IntelliSense** — JSON schema completions for all F5 XC resource types
- **Multi-cloud** — integrates with AWS, Azure, GCP, GitHub, GitLab, Terraform,
  and Salesforce
