## การตรวจสอบ Schema

ส่วนขยายมี JSON Schema สำหรับประเภททรัพยากรทั่วไป เมื่อแก้ไขไฟล์ JSON ของทรัพยากร VS Code จะตรวจสอบการกำหนดค่าของคุณแบบเรียลไทม์และไฮไลต์ข้อผิดพลาด

## Schema ที่รองรับ

- HTTP Load Balancer (`.http_loadbalancer.json`)
- Origin Pool (`.origin_pool.json`)
- App Firewall / WAF (`.app_firewall.json`)
- Health Check (`.healthcheck.json`)
- TCP Load Balancer (`.tcp_loadbalancer.json`)
- DNS Load Balancer (`.dns_load_balancer.json`)
- Virtual Site (`.virtual_site.json`)

## การเติมข้อความอัตโนมัติ

IntelliSense มีการเติมข้อความอัตโนมัติสำหรับชื่อฟิลด์ ค่า enum และโครงสร้างออบเจกต์แบบซ้อนกัน โดยอิงตามข้อกำหนด OpenAPI ของ F5 XC
