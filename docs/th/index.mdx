F5 Distributed Cloud Tools คือ VS Code ส่วนขยายที่ช่วยให้คุณสามารถเรียกดู สร้าง แก้ไข และลบทรัพยากร F5 XC ได้โดยตรงจากโปรแกรมแก้ไขของคุณ โดยรองรับการดำเนินการ CRUD อย่างครบถ้วน รวมถึง JSON IntelliSense พร้อมการตรวจสอบสคีมา เครื่องมือการสังเกตการณ์ และการตรวจสอบสถานะคลาวด์สำหรับทรัพยากรมากกว่า 236 ประเภท ครอบคลุม 25 โดเมน API
