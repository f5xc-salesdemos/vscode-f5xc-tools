F5 Distributed Cloud Tools คือ VS Code ส่วนขยายที่ช่วยให้คุณสามารถเรียกดู สร้าง แก้ไข และลบทรัพยากร F5 XC ได้โดยตรงจากตัวแก้ไขโค้ดของคุณ รองรับการดำเนินการ CRUD อย่างครบถ้วน พร้อม JSON IntelliSense ที่มีการตรวจสอบ schema เครื่องมือสำหรับการสังเกตการณ์ และการติดตามสถานะคลาวด์สำหรับทรัพยากรมากกว่า 236 ประเภทใน 25 โดเมน API
