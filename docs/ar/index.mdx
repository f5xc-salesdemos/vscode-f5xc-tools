أدوات F5 Distributed Cloud هي إضافة VS Code تُمكّنك من استعراض موارد F5 XC وإنشائها وتعديلها وحذفها مباشرةً من المحرر. توفر عمليات CRUD الكاملة، وIntelliSense بصيغة JSON مع التحقق من صحة المخطط، وأدوات قابلية المراقبة، ومراقبة حالة السحابة لأكثر من 236 نوعًا من الموارد عبر 25 نطاقًا من نطاقات API.
