## التحقق من صحة المخطط

توفر الإضافة مخططات JSON لأنواع الموارد الشائعة. عند تحرير ملفات JSON للموارد، يتحقق VS Code من صحة التكوين في الوقت الفعلي ويبرز الأخطاء.

## المخططات المدعومة

- موازن تحميل HTTP (`.http_loadbalancer.json`)
- مجموعة المصدر (`.origin_pool.json`)
- جدار حماية التطبيق / جدار حماية تطبيقات الويب (WAF) (`.app_firewall.json`)
- فحص الصحة (`.healthcheck.json`)
- موازن تحميل TCP (`.tcp_loadbalancer.json`)
- موازنة تحميل DNS (`.dns_load_balancer.json`)
- الموقع الافتراضي (`.virtual_site.json`)

## الإكمال التلقائي

يوفر IntelliSense إكمالاً تلقائياً لأسماء الحقول وقيم التعداد وهياكل الكائنات المتداخلة استناداً إلى مواصفة F5 XC OpenAPI.
