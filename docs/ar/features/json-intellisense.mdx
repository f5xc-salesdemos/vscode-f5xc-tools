## التحقق من المخطط

توفر الإضافة مخططات JSON لأنواع الموارد الشائعة. عند تحرير ملفات JSON للموارد، يتحقق VS Code من صحة تكوينك في الوقت الفعلي ويسلط الضوء على الأخطاء.

## المخططات المدعومة

- موازن تحميل HTTP (`.http_loadbalancer.json`)
- مجموعة خوادم المصدر (`.origin_pool.json`)
- جدار حماية التطبيقات / WAF (`.app_firewall.json`)
- فحص الصحة (`.healthcheck.json`)
- موازن تحميل TCP (`.tcp_loadbalancer.json`)
- موازنة تحميل DNS (`.dns_load_balancer.json`)
- الموقع الافتراضي (`.virtual_site.json`)

## الإكمال التلقائي

يوفر IntelliSense إكمالاً تلقائياً لأسماء الحقول وقيم التعداد وهياكل الكائنات المتداخلة استناداً إلى مواصفات OpenAPI الخاصة بـ F5 XC.
