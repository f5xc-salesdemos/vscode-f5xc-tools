## スキーマ検証

この拡張機能は、一般的なリソースタイプ向けの JSON スキーマを提供します。リソース JSON ファイルの編集中、VS Code はリアルタイムで設定を検証し、エラーをハイライト表示します。

## サポートされているスキーマ

- HTTP ロードバランサー (`.http_loadbalancer.json`)
- オリジンサーバープール (`.origin_pool.json`)
- App Firewall / WAF (`.app_firewall.json`)
- ヘルスチェック (`.healthcheck.json`)
- TCP ロードバランサー (`.tcp_loadbalancer.json`)
- DNS ロードバランシング (`.dns_load_balancer.json`)
- バーチャルサイト (`.virtual_site.json`)

## 自動補完

IntelliSense は、F5 XC OpenAPI 仕様に基づいて、フィールド名、列挙値、およびネストされたオブジェクト構造の自動補完を提供します。
