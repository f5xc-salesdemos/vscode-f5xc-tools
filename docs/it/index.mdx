F5 Distributed Cloud Tools è una VS Code Estensione che consente di esplorare, creare, modificare ed eliminare risorse F5 XC direttamente dall'editor. Fornisce operazioni CRUD complete, IntelliSense JSON con validazione dello schema, strumenti di Osservabilità e monitoraggio dello stato del cloud per oltre 236 tipi di risorse distribuiti su 25 domini API.
