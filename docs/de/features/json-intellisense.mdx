## Schema-Validierung

Die Erweiterung stellt JSON-Schemata für gängige Ressourcentypen bereit. Beim Bearbeiten von Ressourcen-JSON-Dateien validiert VS Code Ihre Konfiguration in Echtzeit und hebt Fehler hervor.

## Unterstützte Schemata

- HTTP-Lastverteiler (`.http_loadbalancer.json`)
- Ursprungsserver-Pool (`.origin_pool.json`)
- App-Firewall / WAF (`.app_firewall.json`)
- Integritätsprüfung (`.healthcheck.json`)
- TCP-Lastverteiler (`.tcp_loadbalancer.json`)
- DNS-Lastverteilung (`.dns_load_balancer.json`)
- Virtueller Standort (`.virtual_site.json`)

## Automatische Vervollständigung

IntelliSense bietet automatische Vervollständigung für Feldnamen, Enum-Werte und verschachtelte Objektstrukturen basierend auf der F5 XC OpenAPI-Spezifikation.
