F5 Distributed Cloud Tools ist eine VS Code Erweiterung, mit der Sie F5 XC-Ressourcen direkt aus Ihrem Editor heraus durchsuchen, erstellen, bearbeiten und löschen können. Sie bietet vollständige CRUD-Operationen, JSON IntelliSense mit Schema-Validierung, Beobachtbarkeitswerkzeuge und Cloud-Statusüberwachung für mehr als 236 Ressourcentypen über 25 API-Domänen.
