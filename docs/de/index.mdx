F5 Distributed Cloud Werkzeuge ist eine VS Code Erweiterung, mit der Sie F5 XC-Ressourcen direkt aus Ihrem Editor durchsuchen, erstellen, bearbeiten und löschen können. Sie bietet vollständige CRUD-Operationen, JSON-IntelliSense mit Schema-Validierung, Beobachtbarkeitswerkzeuge sowie Cloud-Statusüberwachung für mehr als 236 Ressourcentypen in 25 API-Domänen.
