F5 分散式雲端工具是一款 VS Code 擴充功能，讓您能夠直接在編輯器中瀏覽、建立、編輯及刪除 F5 XC 資源。它提供完整的 CRUD 操作、具備結構描述驗證的 JSON IntelliSense、可觀測性工具，以及涵蓋 25 個 API 網域、超過 236 種資源類型的雲端狀態監控功能。
