F5 Distributed Cloud Tools es una VS Code Extensión que le permite explorar, crear, editar y eliminar recursos de F5 XC directamente desde su editor. Ofrece operaciones CRUD completas, IntelliSense de JSON con validación de esquemas, herramientas de Observabilidad y supervisión del estado de la nube para más de 236 tipos de recursos en 25 dominios de API.
