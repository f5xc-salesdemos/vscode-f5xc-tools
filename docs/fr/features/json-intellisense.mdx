## Validation de schéma

L'extension fournit des schémas JSON pour les types de ressources courants. Lors de la modification de fichiers JSON de ressources, VS Code valide votre configuration en temps réel et met en évidence les erreurs.

## Schémas pris en charge

- Équilibreur de charge HTTP (`.http_loadbalancer.json`)
- Pool d'origine (`.origin_pool.json`)
- Pare-feu applicatif / WAF (`.app_firewall.json`)
- Contrôle de santé (`.healthcheck.json`)
- Équilibreur de charge TCP (`.tcp_loadbalancer.json`)
- Équilibrage de charge DNS (`.dns_load_balancer.json`)
- Site virtuel (`.virtual_site.json`)

## Auto-complétion

IntelliSense fournit l'auto-complétion pour les noms de champs, les valeurs d'énumération et les structures d'objets imbriqués sur la base de la spécification OpenAPI F5 XC.
