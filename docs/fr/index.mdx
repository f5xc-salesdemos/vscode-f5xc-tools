F5 Distributed Cloud Tools est une extension VS Code qui vous permet de parcourir, créer, modifier et supprimer des ressources F5 XC directement depuis votre éditeur. Elle offre des opérations CRUD complètes, IntelliSense JSON avec validation de schéma, des outils d'Observabilité, ainsi qu'une surveillance de l'état du cloud pour plus de 236 types de ressources répartis sur 25 domaines API.
