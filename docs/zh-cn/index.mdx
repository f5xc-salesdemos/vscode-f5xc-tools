F5 分布式云工具是一款 VS Code 扩展，使您能够直接在编辑器中浏览、创建、编辑和删除 F5 XC 资源。它提供完整的 CRUD 操作、带模式验证的 JSON IntelliSense、可观测性工具，以及针对 25 个 API 域中 236+ 种资源类型的云状态监控。
