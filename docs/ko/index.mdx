F5 Distributed Cloud 도구는 편집기에서 직접 F5 XC 리소스를 탐색, 생성, 편집 및 삭제할 수 있도록 하는 VS Code 확장입니다. 25개의 API 도메인에 걸쳐 236개 이상의 리소스 유형에 대해 완전한 CRUD 작업, 스키마 유효성 검사를 포함한 JSON IntelliSense, 관측 가능성 도구 및 클라우드 상태 모니터링을 제공합니다.
