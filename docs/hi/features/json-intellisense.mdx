## स्कीमा सत्यापन

एक्सटेंशन सामान्य संसाधन प्रकारों के लिए JSON स्कीमा प्रदान करता है। संसाधन JSON फ़ाइलों को संपादित करते समय, VS Code आपके कॉन्फ़िगरेशन को वास्तविक समय में सत्यापित करता है और त्रुटियों को हाइलाइट करता है।

## समर्थित स्कीमा

- HTTP लोड बैलेंसर (`.http_loadbalancer.json`)
- ऑरिजिन पूल (`.origin_pool.json`)
- ऐप फ़ायरवॉल / WAF (`.app_firewall.json`)
- हेल्थ चेक (`.healthcheck.json`)
- TCP लोड बैलेंसर (`.tcp_loadbalancer.json`)
- DNS लोड बैलेंसर (`.dns_load_balancer.json`)
- वर्चुअल साइट (`.virtual_site.json`)

## ऑटो-पूर्णता

IntelliSense, F5 XC OpenAPI स्पेसिफिकेशन के आधार पर फ़ील्ड नामों, enum मानों और नेस्टेड ऑब्जेक्ट संरचनाओं के लिए ऑटो-पूर्णता प्रदान करता है।
