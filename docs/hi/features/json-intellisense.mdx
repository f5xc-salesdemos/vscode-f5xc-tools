## स्कीमा सत्यापन

एक्सटेंशन सामान्य संसाधन प्रकारों के लिए JSON स्कीमा प्रदान करता है। संसाधन JSON फ़ाइलों को संपादित करते समय, VS Code आपकी कॉन्फ़िगरेशन को रियल-टाइम में सत्यापित करता है और त्रुटियों को हाइलाइट करता है।

## समर्थित स्कीमा

- HTTP लोड बैलेंसर (`.http_loadbalancer.json`)
- ऑरिजिन सर्वर पूल (`.origin_pool.json`)
- ऐप फ़ायरवॉल / वेब ऐप फ़ायरवॉल (WAF) (`.app_firewall.json`)
- हेल्थ चेक (`.healthcheck.json`)
- TCP लोड बैलेंसर (`.tcp_loadbalancer.json`)
- DNS लोड बैलेंसिंग (`.dns_load_balancer.json`)
- वर्चुअल साइट (`.virtual_site.json`)

## ऑटो-कंप्लीशन

IntelliSense, F5 XC OpenAPI स्पेक के आधार पर फ़ील्ड नामों, enum मानों और नेस्टेड ऑब्जेक्ट संरचनाओं के लिए ऑटो-कंप्लीशन प्रदान करता है।
