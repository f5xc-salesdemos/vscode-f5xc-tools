F5 Distributed Cloud उपकरण एक VS Code एक्सटेंशन है जो आपको अपने एडिटर से सीधे F5 XC संसाधनों को ब्राउज़ करने, बनाने, संपादित करने और हटाने में सक्षम बनाता है। यह 25 API डोमेन में 236+ संसाधन प्रकारों के लिए पूर्ण CRUD ऑपरेशन, JSON IntelliSense तथा स्कीमा सत्यापन, अवलोकनीयता उपकरण और क्लाउड स्थिति निगरानी प्रदान करता है।
