## Validação de Schema

A extensão fornece schemas JSON para tipos de recursos comuns. Ao editar arquivos JSON de recursos, o VS Code valida sua configuração em tempo real e destaca os erros.

## Schemas Suportados

- HTTP Load Balancer (`.http_loadbalancer.json`)
- Origin Pool (`.origin_pool.json`)
- App Firewall / WAF (`.app_firewall.json`)
- Health Check (`.healthcheck.json`)
- TCP Load Balancer (`.tcp_loadbalancer.json`)
- DNS Load Balancer (`.dns_load_balancer.json`)
- Virtual Site (`.virtual_site.json`)

## Preenchimento Automático

O IntelliSense fornece preenchimento automático para nomes de campos, valores de enumeração e estruturas de objetos aninhados com base na especificação OpenAPI do F5 XC.
