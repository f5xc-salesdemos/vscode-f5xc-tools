# xcsh for VS Code

> VS Code에서 직접 F5 Distributed Cloud 리소스를 관리하세요

![xcsh](resources/screenshot-welcome.png)

## 시작하기

1. **확장 설치** — VS Code 확장 패널에서 "xcsh"를 검색하세요
2. **xcsh 설치** — `brew install f5xc-salesdemos/tap/xcsh`
3. **컨텍스트 추가** — 명령 팔레트(`Cmd+Shift+P`)를 열고 **xcsh: Add Context**를
   실행하세요

## 주요 기능

- **리소스 탐색 및 관리** — 사이드바에서 로드 밸런서, WAF 정책, 오리진 풀 등을
  생성, 편집, 삭제할 수 있습니다
- **AI 채팅 어시스턴트** — `@xcsh`에 자연어로 질문하여 플랫폼을 관리하세요
- **클라우드 상태 대시보드** — 글로벌 인프라 상태를 실시간으로 한눈에 확인할 수
  있습니다
- **IntelliSense** — 모든 F5 XC 리소스 유형에 대한 JSON 스키마 자동 완성을
  제공합니다
- **멀티 클라우드** — AWS, Azure, GCP, GitHub, GitLab, Terraform, Salesforce와
  통합됩니다
