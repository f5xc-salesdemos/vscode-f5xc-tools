# xcsh for VS Code

> 直接在 VS Code 中管理 F5 Distributed Cloud 资源

![xcsh](resources/screenshot-welcome.png)

## 快速开始

1. **安装扩展** — 在 VS Code 扩展面板中搜索 "xcsh"
2. **安装 xcsh** — `brew install f5xc-salesdemos/tap/xcsh`
3. **添加上下文** — 打开命令面板（`Cmd+Shift+P`）并运行 **xcsh: Add Context**

## 功能概览

- **浏览和管理资源** — 从侧边栏创建、编辑、删除负载均衡器、WAF 策略、源站池等
- **AI 聊天助手** — 用自然语言向 `@xcsh` 提问来管理您的平台
- **云状态仪表板** — 实时全球基础设施健康状态一览
- **IntelliSense** — 所有 F5 XC 资源类型的 JSON Schema 自动补全
- **多云集成** — 支持 AWS、Azure、GCP、GitHub、GitLab、Terraform 和 Salesforce
